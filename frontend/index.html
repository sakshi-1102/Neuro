About NeuroVoice

Deep Learning Based Brain-Computer Interface System

Project Overview

NeuroVoice is a web-based Brain-Computer Interface system for recognition of covert speech to augment communication for paralyzed patients. It uses EEGNet deep learning model to decode imagined words in real time.

ModelEEGNet (CNN-based)
Signal typeEEG + fNIRS (hybrid)
ClassesYes, No, Help, Pain, Water, Toilet, Doctor, Family
LanguagesEnglish, Hindi, Marathi TTS
StackPython · TensorFlow · FastAPI · SQLite · HTML/CSS/JS
DeveloperM.Tech Project — 2026
